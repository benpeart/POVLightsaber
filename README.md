# POV Lightsaber
A Persistance of Vision light saber

# MVP Features
* Can display built in images
* Turn on/off light saber in response to button press
* Animate light saber turning on/off
* Can display any set color of blade (good vs evil)
* Makes sounds when turned on/off
* Makes sounds when moving
* Makes sounds when something is struck

# V.Next Features
* Monitor battery level and warn/power down when getting low (<10V)
* Animated light saber colors
* Power button color matches light saber color
* WebUI to allow changing color/brightness of light saber
* WebUI to allow uploading new images
